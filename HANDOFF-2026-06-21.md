## Critical business rules (never change)

### Attendance
- Worker reported at Site A → removed from Site A queue, appears in Site B with colored dot
- Transfer → `SiteTransferEvent` + push to owner
- Transfer ≠ Late: reported A before 08:00, B after 08:00 → NOT late

### Half-day (locked 2026-06-15)
- Morning: arrived before 12:00 only → auto-detected
- Afternoon: 12:00–12:30 = no late; 12:30–13:00 = late + half day; after 13:00 = exception

### Rain
- Site-scoped, not global
- Can only be set if no reports yet OR owner decides (with wage decision panel)
- Rain wage decision blocks daily report until decided

### Event sourcing
- Every action = event. Originals never deleted. Corrections require a note.

### Timezone
- Always Asia/Bangkok. Server time = source of truth.
